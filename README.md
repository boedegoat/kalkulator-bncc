# Kalkulator BNCC

```
Kalkulator BNCC
Made with love by Bhremada Fevreano

Format: <angka1> <operator> <angka2>, Contoh: 1 + 1
List operator:
  + : Penjumlahan
  - : Pengurangan
  x : Perkalian
  / : Pembagian
  % : Modulus
Ketik q untuk keluar

> 1 + 1
  2
> 5 / 2
  2.5
> bncc
  Input tidak valid
> 10 / 0
  Tidak bisa dibagi 0
```
